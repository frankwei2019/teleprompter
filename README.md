# 词悬浮

轻量级桌面提词器，悬浮置顶、永远在前，专为录视频/直播设计。

## 下载安装

在 [Releases](https://github.com/frankwei2019/teleprompter/releases) 页面下载安装包：
- `.msi` — Windows Installer 格式
- `-setup.exe` — NSIS 安装程序

## 核心功能

- **多预设脚本**：可保存多个提示词文稿，一键切换
- **可调滚动**：滚动速度、字体大小实时可调
- **外观自定义**：背景透明度、字体颜色（7 种）
- **镜像模式**：适配提词器玻璃（左右翻转）
- **循环播放**：到末尾自动回到开头继续滚动
- **3 秒倒计时**：按下播放/重置后倒计时 3 秒再开始，便于就位
- **文件导入**：支持 `.txt` / `.md` 文件
- **状态自动保存**：所有设置和文稿本地持久化
- **悬浮置顶**：始终在最前，不被其他窗口遮挡

## 使用说明

### 窗口操作
- 拖动顶部黑条可移动窗口
- 右上角 `×` 关闭程序
- `?` 查看软件信息和构建日期
- `⚙` 展开/收起设置面板

### 控制栏（窗口底部）
- `↺ 重置` — 倒计时 3 秒后从顶部开始滚动
- `⏮` / `⏭` — 上一个 / 下一个预设（自动播放）
- `▶` / `⏸` — 播放 / 暂停

### 设置面板

| 选项 | 范围 | 说明 |
|------|------|------|
| 滚动速度 | 5–200 | 数值越大滚动越快 |
| 字体大小 | 16–120 | 像素 |
| 背景透明 | 0–100% | 0% 全黑，100% 全透明 |
| 字体颜色 | 7 色 | 蓝、绿、黄等 |
| 镜像模式 | 开/关 | 配合提词器玻璃使用 |
| 循环播放 | 开/关 | 关闭则到尾停止 |

### 导入文稿
1. 在设置面板「文本」区点击「导入文件」
2. 选择 `.txt` 或 `.md` 文件
3. 文稿自动填入当前预设

## 系统要求
